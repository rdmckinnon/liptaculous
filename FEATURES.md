# Liptaculous Coming Soon Page - Feature Overview

## Design Elements

### Color Palette
The site uses a sophisticated, feminine color scheme:
- **Cherry Blossom Pink** (#f4acb7) - Primary brand color
- **Pink** (#ffcad4) - Accent color for CTAs
- **Champagne Pink** (#ffe5d9) - Warm neutral backgrounds
- **Platinum** (#d8e2dc) - Cool neutral for contrast
- **Mountbatten Pink** (#9d8189) - Text and subtle elements

### Typography
- **Playfair Display** - Elegant serif font for headers and brand name
- **Poppins** - Clean sans-serif for body text and UI elements

### Animated Elements

#### Floral Decorations
Three decorative flowers positioned around the page:
- Each flower has 5 petals arranged in a radial pattern
- Gentle rotation animation (20s cycle)
- Fade-in animation on page load
- Positioned strategically to frame the content

#### Leaf Elements
Three decorative leaves scattered across the page:
- Created using SVG paths for organic shapes
- Various rotations and sizes for visual interest
- Subtle fade-in animations with staggered timing
- Low opacity for background effect

### Interactive Features

#### Email Signup Form
- Rounded pill-shaped input and button
- Smooth focus states with color transitions
- Form validation (email required)
- Ready for Netlify Forms integration
- Privacy reassurance message

#### Social Media Links
- Instagram, TikTok, and Facebook icons
- Hover effects with color change and lift animation
- SVG icons for crisp rendering at any size
- Easily customizable links

### Responsive Design
- Fluid typography using clamp() for perfect scaling
- Mobile-first approach
- Stacked layout on small screens
- Full-width form fields on mobile
- Touch-friendly button sizes

### Performance Optimizations
- Static site generation for fast loading
- Inline CSS for critical styles
- Optimized SVG graphics
- Google Fonts preconnect for faster font loading
- No JavaScript required for core functionality

## User Experience

### Visual Flow
1. Brand name and tagline capture attention
2. "Coming Soon" message sets expectations
3. Compelling description explains the value proposition
4. Clear call-to-action with email signup
5. Social media links for early engagement

### Emotional Impact
- Elegant and luxurious feel
- Feminine without being overly pink
- Sophisticated color combinations
- Animated elements add life and interest
- Clean, uncluttered layout maintains focus
